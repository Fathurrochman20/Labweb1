### Instruksi Praktikum
1. Persiapkan text editor
   misalnya VSCode.
2. Buat file baru dengan nama
   lab1_tag_dasar.html
3. Buat struktur dasar
   dari dokumen HTML.

   akan dijelaskan berikutnya.

   html dengan mengakses
   http://validator.w3.org
